# Monitor Platform Backend

NestJS backend for the frontend monitoring platform.

- **Error collection** — receives JS/Promise/React errors from the SDK
- **Source Map resolution** — restores minified stack traces to original source
- **AI root-cause analysis** — LangGraph pipeline powered by OpenRouter LLM
